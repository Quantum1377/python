# Telegram Sales Bot

Bot simples em Python para enviar notificações de vendas e checar estoque.

## Funcionalidades

- `/start` → Mensagem de boas-vindas
- `/vendas` → Lista todos os produtos com estoque e preço
- `/estoque <produto>` → Checa estoque de um produto específico
